# RagdollMod
A mod for Gorilla Tag that lets you ragdoll like Garry's Mod.

<img src="https://www.image2url.com/r2/default/gifs/1780423272945-a2fce340-e3ba-4f66-8771-7c8a3ff18ef3.gif">

---

# Controls
- Open Settings: P **(PC)**
- Toggle Death: Right Joystick Click **(VR)** / B **(PC)**

---

# Installation

- Download the release from [here](https://github.com/InoxiGtag/RagdollMod/releases/latest)
- Drag the file `RagdollMod.dll` to your plugins folder
- Start Gorilla Tag if needed

---

## Settings
- Ragdoll Velocity
- Ragdoll Free Cam
